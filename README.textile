h1. Lmstats

This is a small Rails application for storing floating license usage statistics.
It was developed by Giovanni Cangiani for internal use at
"Kandou Bus":http://www.kandou.com/
*Use it at your own risk!*.

h2. Requirements

This application requires:

* Ruby version 1.9.3
* Rails version 4.0.0
* An SQL server (MySQL or PostGres)
* One or more servers running FlexLM License Manager

Learn more about "Installing Rails":http://railsapps.github.io/installing-rails.html.
